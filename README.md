# Fantasy Football Trade Optimization Model

## Overview
This project analyzes fantasy football player performance to identify potential undervalued and overvalued players. The goal was to combine predictive modeling, time-based forecasting, and optimization logic to generate data-driven trade recommendations.

## Objective
The main objective was to help fantasy football managers make better trade decisions by comparing projected player value against expected performance.

## Methods
- Collected and organized weekly fantasy football player data
- Built regression models to estimate future player performance
- Used lagged variables to account for previous-week performance trends
- Compared projected PPR points with actual PPR outcomes
- Applied optimization logic to evaluate potential trade scenarios

## Tools Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Excel Solver / Optimization concepts

## Key Skills Demonstrated
- Data cleaning
- Exploratory data analysis
- Predictive modeling
- Regression analysis
- Forecasting
- Optimization modeling
- Data visualization
- Sports analytics

## Results
The model helped identify players whose market value differed from their projected fantasy performance. Trade recommendations were generated by comparing expected value, player rankings, and lineup needs.
